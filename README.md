# SmartCard Guard

Turn any smart card, ATM card, or employee ID into a physical Linux workstation key.

## Features

- Smart card presence detection
- Automatic workstation lock on card removal
- Continuous access enforcement
- KDE Plasma support
- Wayland compatible
- Lightweight Bash implementation
- Open Source

## Security Notice

SmartCard Guard does not read banking information, PINs, account numbers, or cardholder data.

The smart card is only used as a physical presence token.

## Installation

Download:

```bash
chmod +x install.sh
./install.sh
```

## Uninstall

```bash
chmod +x uninstall.sh
./uninstall.sh
```
